The border-left Property

This property is a shorthand property for border-left-width,
border-left-style, and border-left-color. content-box: This is the default value. In this case, the
width and height of an element are calculated based only on the content of the element. Padding and border are added to the width and height of the element, which can cause the overall size of the element to be larger than expected. border-box: In this case, the width and height of an element are calculated based on the content, padding, and border of the element. This means that the overall size of the element includes the padding and border, and the content area is adjusted to fit inside
